# Amazon Clone Website 🛒

This is a front-end clone of the Amazon website built using **HTML and CSS**.  
It mimics the homepage UI of Amazon with responsive layout, navigation bar, search bar, and product sections.

## 🔧 Technologies Used
- HTML5
- CSS3
- Font Awesome
